髮帳 (HairBook) 隱私權政策

最近更新日期：2026年2月3日

  1. 資料收集與來源

    我們在提供服務時，會收集以下用戶資料：

    • 基本帳號資訊（姓名、Email、來自 Google/LINE 的第三方登入識別碼）
    • 預約紀錄與服務偏好
    • 照片或影像檔（用於服務紀錄與預約管理）
    • 使用行為分析資料（透過 Firebase Analytics 收集，不含個資）
    • Google 日曆資料： 若您授權同步功能，我們將存取您的 Google 日曆事件資訊。

  2. Google 用戶數據專屬聲明 (Google API Disclosure)

    針對透過 Google API 取得的資訊，髮帳 承諾遵守以下規範：

    • 數據使用： 我們僅存取您的 Google 日曆以實現「自動同步預約行程」功能。我們不會將這些資料用於任何其他目的。
    • 有限使用 (Limited Use)： 髮帳 對從 Google API 接收資訊的使用與轉移，均遵循 Google API 服務用戶數據政策，包括其中的「有限使用」要求。
    • 數據分享與披露： 我們不會將您的 Google 用戶數據分享、轉移或披露給任何第三方，除非是為了提供或改進本應用的預約功能、或是為了遵守法律要求。我們絕對不會將您的 Google 數據用於廣告投放、出售給第三方，或用於信貸評估。

  3. 資料加密與保護

    為保障用戶資料安全，我們採取以下措施：

    • 敏感個人資料於伺服器端進行 AES-256-CBC 加密儲存。
    • 影像檔案儲存於 Cloudflare R2，並具備伺服器端加密保護。
    • 系統僅授權必要之程式或受訓管理人員存取相關資料。

  4. 用戶權益與資料刪除

    用戶可隨時透過 App 查詢、更新或要求刪除個人資料。當用戶要求刪除帳號時，我們將於合理時間內從主資料庫中清除所有關聯資料（包括已同步之 Google 日曆緩存資訊）。

  5. 條款更新

    隱私權政策如有重大更新，我們會在 App 內通知用戶。繼續使用服務即表示您同意更新後的條款。
